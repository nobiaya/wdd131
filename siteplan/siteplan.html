Site Name

Sight Explorer

This name was chosen because it clearly communicates the main function of the website - it displays pictures of places I have visited and provides the exact location with a map direction when an image is clicked.

Site Purpose

The purpose of *Sight Explorer* is to allow users to explore the different places I have visited by viewing images and extracting location details. Users will be able to:

  • View a gallery of places I've visited.
  • Click on images icon to access the exact location on a map.
  • Learn more about each place through descriptions and additional information.

The goal is to share my travel experiences and inspire others to explore these locations through an interactive gallery and map feature.

Scenarios

  1. How can I view the exact location of a place I see in the gallery?
  2. What information is provided about each location in the gallery?
  3. How can I learn more about the places featured on the *Sight Explorer* site?

The primary color (hsl(34, 90%, 55%)) will be used for main headings and interactive elements like buttons.

The secondary color (hsl(210, 36%, 96%)) serves as the background color for the site, providing a clean and neutral base.

The text color (hsl(0, 0%, 20%)) ensures good readability against the light background.

Typography

Headings: Playfair Display, serif

Body Text: Roboto, sans-serif

This combination provides a modern, elegant look while ensuring readability across devices.
